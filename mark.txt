
从源码安装
    git clone https://github.com/openclaw/openclaw.git
    cd openclaw
    pnpm install && pnpm build && pnpm ui:build
    pnpm link --global
    pnpm openclaw onboard --install-daemon

在浏览器中打开 Control UI
 pnpm openclaw dashboard

 启动后端管理台
  pnpm ui-admin:dev


  ui目录修改后一定要执行（ pnpm build 不能省略）
  pnpm build && pnpm ui:build




后台管理做了开机自启
systemctl restart openclaw-ui-admin.service
/etc/systemd/system/openclaw-ui-admin.service #服务写在了这里

---------------------------------------------
你要用局域网打开 主网关 18789
在主配置 openclaw.json（常见为 ~/.openclaw/openclaw.json）里增加或修改，例如：

"gateway": {
  "bind": "lan",
  "port": 18789,
  "auth": {
    "mode": "token",
    "token": "你的随机长密钥"
  }
}
bind 设为 lan（或 auto）后，才会在局域网网卡上监听。
还要处理的配置
gateway.controlUi.dangerouslyAllowHostHeaderOriginFallback: true
gateway.controlUi.dangerouslyDisableDeviceAuth: true
